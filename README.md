# Manufacturing Downtime Cost Analysis

## Overview
Analysis of a synthetic manufacturing downtime dataset (5,200 events across 4 production lines, 6 months) using SQL (Google BigQuery) and Tableau, informed by prior operations/project management experience.

## Data
Synthetic dataset modeled on realistic manufacturing downtime patterns — event-level records with cause, line, shift, cost impact, and operator attribution.
